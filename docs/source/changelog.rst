Changelog

ETLPlus publishes release artifacts from Git tags, and the historical release
notes are currently maintained on GitHub Releases.

Documentation Publishing
------------------------

The published docs site now includes:

- getting-started pages for installation and quickstart usage
- guide pages for examples, pipeline authoring, file-handler coverage, and
  contributor expectations
- an autogenerated API reference sourced from ``etlplus`` docstrings
- CI validation for the HTML and EPUB formats advertised to Read the Docs

If you want a full in-repository changelog later, this page is the right place
to migrate release notes out of GitHub Releases.
